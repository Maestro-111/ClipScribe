# Module: Output Parsing

## Purpose
Takes the raw structured metadata generated by `InformationExtractor` (JSON dictionaries) and formats it for specific downstream use cases.

## Key Files
* `parser_core.py`: Base class for taking extraction results and transforming them.
* `youtube/`: Specific parsing logic intended for YouTube video metadata generation or analysis.

## Guidelines
* The Extractor outputs pure factual data. The Parser's job is to apply business logic or formatting to that data.
